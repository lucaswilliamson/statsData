B1

We were able to create a function called getReportCard that returned the final grade and picture for a given penguin in the course. This function relied on previously defined functions to carry out each of the required operations to calculate the weighted average based on the syllabus. Firstly, we created a separate function for each grade category that took a given penguin and returned the value of the mean category grade multiplied by the category weight. Each of these functions in turn relied on a function defined in part A that returned a penguin's category average given a penguin. For the sake of an example, here is the function that returned the weighted test category grade for a penguin, identified as calcTests: 

var calcTests= function(penguin){
    var testsReport= getPenguinTestGrade(penguin);
    return testsReport*.3;
};
In the calcTests function, a variable called testsReport is defined as the value returned by the previously defined getPenguinTestGrade given the inputted penguin; getPenguinTestGrade calculates the mean test grade of a penguin. The calcTests function then returns the output of the testsReport variable (i.e., the mean test grade of the input penguin) multiplied by the weight of the test grade category, which is .3 as defined by the syllabus. A similar function was created for each of the other grade categories: 
    var calcFinal= function(penguin){
    var finalReport= getPenguinFinalGrade(penguin);
    return finalReport*.35;
};
var calcTests= function(penguin){
    var testsReport= getPenguinTestGrade(penguin);
    return testsReport*.3;
};
    
var calcQuizes= function(penguin){
    var quizesReport= getPenguinGrade(penguin);
    return quizesReport*.2;
};

var calcHomework= function(penguin){
    var homeworkReport= getPenguinHWGrade(penguin);
    return homeworkReport*.15;
};
In the above code, calcFinal returns the weighted final exam grade given a penguin, calcQuizes returns the weighted quiz average given a penguin, and calcHomework returns the weighted homework average given a penguin. Next, we created a function called calculateReport that returns the final average of a given penguin by taking the sum of the weighted grade categories: 
var calculateReport = function(penguin){
    return calcFinal(penguin)+calcTests(penguin)+calcQuizes(penguin)+calcHomework(penguin);
};
Finally, the getReportCard function was defined. This function returns an object containing a penguin's picture and final grade given the penguin: 
var getReportCard= function(penguin) {
    var obj= {
    Pic: getPic(penguin),
    Report: calculateReport(penguin)
    
    }
    return obj;
}
The getReportCard function was then mapped to the entire classroom, which returns an array of objects. Each object in the array contains a particular penguin's picture and final grade: 
var answer= classroom.map(getReportCard);

In order to answer this question, we needed to create a function that returned the first quiz grade of a penguin given the penguin. We called this function getFirstQuiz: 

var getFirstQuiz = function(penguin) {
    return penguin.quizes[0].grade;
};
This function takes a penguin as the input and then returns the value for the grade of the zeroeth object in the quiz array of the penguin. We then wrote a line that maps the getFirstQuiz function to the entire classroom and takes the mean of that value, which returns the average of all the first quiz grades in the classroom: 
var firstQuizMean = d3.mean(classroom.map(getFirstQuiz));
